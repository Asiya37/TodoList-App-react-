🚀 TaskFlow – To-Do List App

TaskFlow is a high-performance and responsive task management application designed to help users efficiently organize their daily activities. Built using modern frontend technologies, it provides a smooth and interactive user experience.

✨ Features

✅ Add, edit, and delete tasks

✔️ Mark tasks as completed

🗑️ Clear completed tasks in one click

📊 Real-time task tracking (total, completed, remaining)

💾 Data persistence using localStorage

📱 Fully responsive design

🛠️ Tech Stack

⚛️ React (Functional Components & Hooks)

⚡ Vite

🎨 HTML, CSS (Flexbox & Responsive Design)

📂 Project Structure

src/
 ├── App.jsx
 ├── components/
 │    ├── TodoList.jsx
 │    └── TodoItem.jsx
